# ICS3U-Tic-Tac-Toe
An advanced Tic Tac Toe game featuring animations, sound effects, music playlist, multiple AI difficulty levels, player statistics, and leaderboards.

## Features
- Multiple difficulty levels (AI opponent)
- Player vs Player
- Music playlist and sound effects
- Animations
- Player statistics and leaderboards
- Clean and intuitive UI

## Technologies Used
- Java
- Swing (GUI Framework)
- AWT (Abstract Window Toolkit)
- 2D Graphics
- Sound API
- I/O (Input / Output)
- Util

## How to Run
1. Clone the repository
2. Run the main file
3. Enjoy the game

- Smarter AI
